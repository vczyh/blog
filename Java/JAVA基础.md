### JAVA基础

##### 八种基本数据类型

| 类型    | 字节 | 范围              | 默认值   |
| ------- | ---- | ----------------- | -------- |
| byte    | 1    | -128~127          | 0        |
| short   | 2    | -2^15~2^15-1      | 0        |
| int     | 4    | -2^31~2^31-1      | 0        |
| long    | 8    | -2^63~2^63-1      | 0L       |
| char    | 2    | 0~2^16-1[0~65535] | '\u0000' |
| float   | 4    |                   | 0.0F     |
| double  | 8    |                   | 0.0D     |
| boolean |      |                   | false    |
